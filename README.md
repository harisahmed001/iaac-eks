# EKS Cluster

This is EKS cluster with manifects of simple go application

## Apply IaaS

```bash
cd Iaac
terraform plan
terraform apply
```

## Usage

```bash
kubectl create -f Manifests/.
```

## Destory IaaS

```bash
cd Iaac
terraform destory
```
